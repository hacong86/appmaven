# SEO Strategy for Prompt Master AI

Based on the app's purpose (AI prompt storage, generation, and studio tools) and current market trends for 2026, here is tailored keyword research to optimize content.

## 🎯 Primary Keywords (High Volume / Moderate Competition)
> Target these in: H1 Title, Meta Title, and First Paragraph

- **AI Prompt Studio** (Core branding - establishes authority)
- **Prompt Generator tool** (Action-oriented - what users are looking for)
- **Midjourney prompt builder** (Specific niche - high intent users)
- **ChatGPT prompt library** (Resource focused - attracts users looking for examples)

## 🚀 "Blue Ocean" Keywords (Low Competition / High Relevance)
> Use these in: Feature lists (H2/H3 tags), Image Alt Text, and Blog content

- **"Best offline prompt manager"** (Specific unique selling point - offline capability)
- **"Context-aware AI prompts"** (Technical/Pro audience)
- **"Prompt engineering for beginners"** (Educational angle)
- **"Save Midjourney parameters"** (Specific pain point solution)
- **"Dynamic prompt templates"** (Feature specific)

## 📝 SEO Content Optimization Plan

### 1. Meta Title Structure
`Prompt Master: AI Prompt Studio | Builder for ChatGPT & Midjourney`
*Why:* Combines brand name with functional keywords (Builder, Studio) and top platforms.

### 2. Meta Description
`Unlock your creative potential with Prompt Master. The ultimate offline prompt library and builder for ChatGPT, Midjourney, Gemini, and Claude. Craft, organize, and customize professional prompts instantly.`
*Why:* Highlights "offline", "library", and major model names to catch search intent for specific tools.

### 3. URL Structure suggestions
- `/apps/prompt-master-ai/` (Clean, includes main keywords)
- `/apps/prompt-master-ai/midjourney-prompts/` (Potential sub-page)

### 4. Image Alt Text Strategy
Instead of just "Screenshot 1", use desriptive alt text:
- "Visual Prompt Gallery for Midjourney art styles"
- "AI Prompt Builder interface with dynamic variables"
- "Prompt Master detailed prompt organization screen"
